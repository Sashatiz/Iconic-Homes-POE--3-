# Iconic Homes — Final POE (Part 3)

## Overview
Iconic Homes is a 5-page website (Home, About, Services, Enquiry, Contact) built as a WEDE Portfolio of Evidence. This repository contains HTML, CSS and JavaScript for all parts (1–3).

## Files
- index.html, about.html, services.html, enquiry.html, contact.html
- css/style.css
- js/scripts.js
- robots.txt
- sitemap.xml
- images/

## Changelog – Part 3
- Added accordion to services page (JS + CSS).
- Implemented gallery with lightbox in index.html.
- Added dynamic project loading to services page.
- Added search/filter for services.
- Added Leaflet map to contact page (OpenStreetMap tiles).
- Implemented client-side validation for enquiry and contact forms.
- Enquiry form shows a cost estimate (R values) and displays results inline.
- Contact form shows an email preview and can open the mail client via `mailto:`.
- Added robots.txt and sitemap.xml for SEO.
- Improved meta descriptions and title tags across pages.
- Added accessibility improvements (alt text, ARIA live where appropriate).
